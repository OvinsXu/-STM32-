## 运行
### 1.Web代码
0. 环境：Node。js
1. 下载源代码后，终端执行`npm install`安装依赖   
2. 使用`node ./app.js`运行服务器程序
### 2.Stm32代码
- 环境:Keil-MDK,Stm32VET6,固件库编程    
- STM32_Codes文件夹代码仅供参考,可自行移植
